# NOVA BLOCKS System Topology

## Status

This document is a planning-oriented overview. It mixes implemented components with aspirational business and infrastructure concepts. Treat revenue, ownership, and regional expansion statements as conceptual unless backed by code, contracts, or audited records.

## Actual Repository Areas

### Application Code

- `nova_blocks/frontend/` - React frontend application
- `nova_blocks/backend/` - Node.js/Express backend application
- `nova_blocks/finance/` - Python-based finance and trading experiments
- `nova_blocks/scripts/` - Benchmark and utility scripts

### Documentation

- `README.md` - Repository summary and setup notes
- `nova_blocks/*.md` - Corporate, revenue, and planning documents
- `docs-site/` - Docusaurus documentation site

## Current Technical Topology

### Frontend

- React single-page application
- Static assets under `nova_blocks/frontend/public/`
- Source code under `nova_blocks/frontend/src/`

### Backend

- Express-based Node.js service
- Jest-based test runner configured at the repository level
- Models, middleware, and scripts under `nova_blocks/backend/`

### Finance and Benchmarking

- Python modules for trading, backtesting, and AI experiments
- Benchmark scripts for Blackwell-related diagnostics and reporting
- Experimental and research-oriented rather than production-hardened

## Planning Notes

## Recommended Interpretation

Use this repository as a source of:
